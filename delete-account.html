Suppression de compte GeoAPS

Les utilisateurs de GeoAPS peuvent supprimer leur compte et les données associées directement depuis l'application.

Suppression depuis l'application

Dans GeoAPS, allez dans :

Paramètres → Supprimer mon compte

Une confirmation est demandée avant suppression. Pour des raisons de sécurité, l'application peut demander le mot de passe du compte. Si le compte est administrateur, un code PIN administrateur peut également être requis.

Données supprimées

La suppression du compte entraîne la suppression des données associées au compte, notamment :

  • le compte utilisateur ;
  • les chantiers enregistrés ;
  • les relevés et points associés ;
  • les paramètres utilisateur ;
  • les données liées à l'utilisation de GeoAPS.

Protection contre les abus

Certaines informations techniques minimales peuvent être conservées afin d'éviter la réinitialisation abusive des essais gratuits après suppression et recréation d'un compte.

Demande par e-mail

Vous pouvez également demander la suppression de votre compte par e-mail à : akkro@outlook.fr

Après validation de la demande, les données concernées seront supprimées dans un délai raisonnable.
