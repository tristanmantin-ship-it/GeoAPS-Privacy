Suppression de compte GeoAPS

Les utilisateurs de GeoAPS peuvent supprimer leur compte et les données associées.

Suppression depuis l'application

La suppression peut être effectuée directement depuis l'application :

Paramètres > Supprimer mon compte

Pour des raisons de sécurité, l'application peut demander le mot de passe du compte. Si le compte est administrateur, un code PIN administrateur peut également être demandé.

Données supprimées

La suppression du compte entraîne la suppression des données associées, notamment :

  • le compte utilisateur ;
  • les chantiers créés ;
  • les données techniques associées aux relevés ;
  • les paramètres personnels enregistrés ;
  • les photographies et données liées au compte lorsque celles-ci sont stockées par l'application.

Données pouvant être conservées

Afin d'éviter le contournement des essais gratuits et de préserver la sécurité de l'application, certaines informations techniques limitées peuvent être conservées après suppression du compte, par exemple un identifiant technique d'appareil ou une empreinte non directement exploitable de l'adresse e-mail.

Demande par e-mail

Vous pouvez également demander la suppression de votre compte par e-mail à : contact@akkroma.fr

Après validation de la demande, les données liées au compte seront supprimées dans un délai raisonnable.
