Politique de confidentialité de GeoAPS

Dernière mise à jour : 2 juillet 2026

Présentation

GeoAPS est une application développée par Akkroma permettant aux professionnels du terrain de réaliser des relevés APS, de la géolocalisation, des métrés, des annotations photographiques et la génération de rapports PDF.

Données collectées

L'application peut collecter les données suivantes :

  • Adresse e-mail utilisée pour l'authentification ;
  • Nom et prénom ;
  • Nom de l'entreprise ;
  • Identifiant technique de l'appareil ;
  • Photographies prises dans l'application ;
  • Coordonnées GPS associées aux relevés ;
  • Données des chantiers créés ;
  • Paramètres utilisateur, clients, fourreaux et annotations personnalisées.

Utilisation des données

Ces données sont utilisées uniquement pour :

  • permettre la connexion à l'application ;
  • gérer les comptes utilisateurs et les droits d'accès ;
  • enregistrer les chantiers et relevés ;
  • générer les rapports PDF ;
  • appliquer les quotas de la version de test ou gratuite ;
  • sécuriser l'accès administrateur ;
  • améliorer le fonctionnement général de GeoAPS.

Stockage et services utilisés

GeoAPS utilise les services Firebase de Google, notamment Firebase Authentication et Cloud Firestore, afin d'assurer l'authentification, la gestion des comptes et le stockage des données nécessaires au fonctionnement de l'application.

Partage des données

Les données ne sont ni vendues ni cédées à des tiers à des fins commerciales. Elles sont utilisées uniquement dans le cadre du fonctionnement de GeoAPS.

Permissions utilisées

  • Appareil photo : prise de photos des relevés terrain ;
  • Localisation : enregistrement des coordonnées GPS des points relevés ;
  • Stockage : génération et partage des rapports PDF.

Sécurité

Les comptes sont protégés par une authentification sécurisée. Certains comptes peuvent être soumis à une validation administrateur. Les comptes administrateurs disposent d'une protection supplémentaire par code PIN.

Conservation des données

Les données sont conservées tant que le compte utilisateur est actif ou jusqu'à leur suppression. Certaines informations techniques peuvent être conservées afin d'éviter les abus liés à la réinitialisation des essais gratuits après suppression d'un compte.

Suppression du compte

La suppression du compte peut être effectuée directement depuis l'application : Paramètres → Supprimer mon compte.

Vous pouvez également consulter la page dédiée : Suppression du compte GeoAPS.

Vos droits

Conformément à la réglementation applicable, vous pouvez demander l'accès, la rectification ou la suppression de vos données personnelles en contactant l'éditeur.
